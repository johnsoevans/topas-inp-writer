# Algorithm & Methodology Papers — Now Summaries, Not PDFs

This skill used to bundle copies of the journal papers behind TOPAS's algorithms. It no longer does — see `references/paper-summaries.md` instead, which has an original summary (problem addressed, method, citation) for each paper, written from reading it rather than reproducing its text.

The one thing worth knowing: a few of these papers carry an IUCr cover page stating the author may self-host that exact reprint on their own web site or institutional repository. That grant covers hosting, not bundling copies into a file that gets handed out to other people, so it doesn't change anything here — nothing in this skill is a reproduction of a published paper's actual text/figures/typeset PDF, regardless of who authored it.

If exact equations, derivations, figures, or data from a specific paper are needed, look it up directly using the citation given in `paper-summaries.md`.
